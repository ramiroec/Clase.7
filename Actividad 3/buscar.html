Métodos de Búsqueda en Cadenas JavaScript

Método indexOf()

Retorna la posición de la primera aparición de un texto especificado.

Sintaxis: cadena.indexOf(textoABuscar, posicionInicio?)

let str = "Por favor, encuentra donde 'encuentra' aparece!";
let posicion = str.indexOf("encuentra");

Método lastIndexOf()

Retorna la posición de la última aparición de un texto especificado.

Sintaxis: cadena.lastIndexOf(textoABuscar, posicionInicio?)

Método includes()

Retorna true si encuentra el texto especificado, false si no.

Sintaxis: cadena.includes(textoABuscar, posicionInicio?)

let texto = "Hola mundo, bienvenido al universo.";
let resultado = texto.includes("mundo");

Método startsWith()

Retorna true si la cadena comienza con el texto especificado, false si no.

Sintaxis: cadena.startsWith(textoABuscar, posicionInicio?)

let texto = "Hola mundo, bienvenido al universo.";
let resultado = texto.startsWith("Hola");

Método endsWith()

Retorna true si la cadena termina con el texto especificado, false si no.

Sintaxis: cadena.endsWith(textoABuscar, longitud?)

let texto = "Hola mundo, bienvenido al universo.";
